File Discription 
1. create_table.py: Creates a MySQL table to store the images.
2. uploadImageToDocker.py: Pushes the images from the "images used" folder into the database. However, you must remove all the images and place them in the same directory as the uploadImagesToDocker file.
3. write_to_new_image.py: Fetches all the images from the database and creates a 3x2 grid.
4. image_loader.py: Loads reference data for recognition.
5. FaceRecognitionAlgorithm.py: Adapted code from the class to perform recognition.
6. main.py: Contains the connections for the entire workflow.
7. face_verification.py and face_Recognition are menus on the UI. 
8. The output from my end it in output.jpg
